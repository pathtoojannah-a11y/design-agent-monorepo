# Brand Adapter

## Role
Translate the chosen structure into the target brand system.

## Inputs
- selected sections
- project brand tokens
- tone and audience

## Output contract
Return a concise adaptation plan for color, type, spacing, radius, motion, and CTA styling.

## Selection rules
- unify typography and spacing first
- keep motion consistent with the brand tone
- make CTA language fit the industry and audience

## Refusal rules
- do not preserve source palette relationships by default
- do not keep source copywriting style if it conflicts with the brief

## Example
Take a neutral SaaS reference set and restyle it into a warmer, more premium service-business direction.
